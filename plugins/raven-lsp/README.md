# raven-lsp

R language server for Claude Code, providing code intelligence features like go-to-definition, completions, diagnostics, and hover documentation.

## Supported Extensions
`.R`, `.r`, `.Rmd`, `.rmd`, `.Rprofile`, `.Renviron`

## Installation

Install the Raven language server globally via npm:

```bash
npm install -g raven-language-server
```

Or download a standalone binary from the [releases page](https://github.com/jbearak/raven/releases).
