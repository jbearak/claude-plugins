# sight-lsp

Stata language server for Claude Code, providing code intelligence features like go-to-definition, completions, diagnostics, and hover documentation.

## Supported Extensions
`.do`, `.ado`, `.doh`, `.mata`

## Installation

Install the Sight language server globally via npm:

```bash
npm install -g sight-language-server
```

Or download a standalone binary from the [releases page](https://github.com/jbearak/sight/releases).
